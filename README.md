## Build System Boilerplate using LESS
Template for front-end development.

Clone or download as zip.

cd into the directory.

run `npm install`

run grunt task `grunt` default to watch less & js

start developing

##Vendor
1. Skeleton
2. Normalize

##LESS Folder Layout
